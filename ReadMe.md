# CFD Test Cases in OpenFOAM: Pipe Flow & Thermal Analysis

A set of test problems for verification and validation of OpenFOAM solvers. The project includes modeling of laminar natural convection, turbulent flow in a pipe, heat transfer with a heat carrier, and temperature distribution in a shell.

## 📁 Project Structure

| Folder | Description |
|--------|-------------|
| `natConvBox/` | Laminar natural convection in square/cubic cavity |
| `pipeTurbulentFlow/` | Turbulent flow in a circular pipe |
| `pipeHeatTransfer/` | Turbulent flow with heat transfer in a pipe |
| `pipeShellThermal/` | Temperature distribution in a pipe shell |
